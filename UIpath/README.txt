UiPath Web Automation – JD Sports Product Data Extraction
Overview
----------------------------------------------------------

This project demonstrates web automation using UiPath Studio to search for products and navigate through multiple pages on the JD Sports website. The automation performs a product search, handles dynamic selectors, and iterates through pagination to extract data from multiple result pages.

This project highlights practical experience in UI automation, selector management, and loop-based navigation.

Website Automated
-----------------

	JD Sports Global:
	https://www.global.jdsports.com/search/tracksuits/?from=72

Features
---------
	- Automated product search using Type Into activity

	- Dynamic selector handling using wildcard (*)

	- Automated pagination using While loop

	- Controlled iteration using Counter variable

	- Stable click automation for Next page navigation

	- Multi-page data extraction

	- Reliable execution with consistent selectors


Key Skills Demonstrated
------------------------
	- Web automation using UiPath

	- Dynamic selector handling

	- Loop implementation and control flow

	- Debugging selector issues

	- Pagination automation

	- Reliable UI interaction automation
