# C# SQL Server Desktop Application

A Windows Forms desktop application built with **C# (.NET 8)** that demonstrates SQL Server integration, database management, and data import functionality.

> **Note:** This project was developed as part of the Udemy course **"Learn C# With Windows Forms and SQL Server"** and extended through hands-on implementation to strengthen practical C# and SQL Server development skills.

## Key Features

- Connect to Microsoft SQL Server
- Configure SQL Server and database connections
- Import CSV data into SQL Server
- View and edit database records
- User-friendly Windows Forms interface
- Database access using ADO.NET (`SqlConnection`, `SqlCommand`)

## Tech Stack

- C#
- .NET 8
- Windows Forms (WinForms)
- Microsoft SQL Server
- ADO.NET
- Visual Studio 2022 Community

## Getting Started

### Prerequisites

- Visual Studio 2022
- .NET 8 SDK
- Microsoft SQL Server

### Run the Project

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Configure the SQL Server connection.
4. Build and run the application.
