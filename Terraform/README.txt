What This Project Does:
-------------------------
- Creates an S3 bucket using Terraform

- Uploads index.html, error.html, and image assets

- Sets public-read permissions for static hosting

- Configures index and error documents

- Produces a working static website URL through S3

Purpose of This Project:
------------------------
- This repository demonstrates:

- Practical IaC (Infrastructure as Code) usage

- Hands-on AWS S3 automation with Terraform

- My learning and progress in DevOps and cloud infrastructure

- A working static website hosted on AWS
